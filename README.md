## Установка
### Скачать данные

Кликнуть на `Code` -> `Download ZIP` чтобы скачать архив. После чего извлечь данные.

### Установка зависимостей

Установить flask с помощью консоли. 
```
pip install flask
```
## Использование

1. Запуск приложения осуществляется также в консоли
```
python weather.py
```
